2次方程式の解説

2次方程式とは、変数 x を含む次の形式の方程式です:

ax2 + bx + c = 0

ここで:

  • a, b, c は定数（a ≠ 0）。
  • x は未知数（解を求める対象）。

解の公式

2次方程式の解は次の公式を使って求めることができます:

x = (-b ± √(b2 - 4ac)) / 2a

ここで:

  • √ は平方根を意味します。
  • b2 - 4ac を「判別式」と呼びます。

判別式による解の性質

判別式 D = b2 - 4ac によって解の性質が決まります:

  • D > 0: 実数解が2つあります。
  • D = 0: 実数解が1つ（重解）です。
  • D < 0: 実数解はなく、虚数解が2つあります。

具体例

例: 方程式 2x2 - 4x + 2 = 0

  1. a = 2, b = -4, c = 2 を確認。
  2. 判別式 D = (-4)2 - 4 × 2 × 2 = 16 - 16 = 0。
  3. D = 0 より、実数解が1つ（重解）。
  4. 解の公式に代入すると x = 1。

このように解を求めます。
